# COMMENT UTILISER CES FICHIERS AVEC CLAUDE CHAT

## Étape 1 — Ouvrir Claude Chat
Aller sur : https://claude.ai

## Étape 2 — Commencer par ce message exact

Copie-colle ce message au début de la conversation :

---

"Bonjour Claude. Je suis Mohamed Yassine Marzouk, étudiant en Master Leadership Managérial 4.0 à la FSJES de Tétouan. J'ai besoin que tu génères mon rapport de PFE complet. Je vais te donner toutes les informations nécessaires dans le fichier joint. Lis-le entièrement avant de commencer."

[Joindre le fichier : BRIEFING_COMPLET.md]

---

## Étape 3 — Demander la génération partie par partie

Claude Chat a une limite de longueur de réponse. Demande le rapport **chapitre par chapitre** :

### Message 1 :
"Génère les pages liminaires : page de garde, dédicace, remerciements, résumé/abstract, liste des abréviations."

### Message 2 :
"Génère l'Introduction générale (3 pages environ)."

### Message 3 :
"Génère le Chapitre 1 complet : Présentation de l'Association El Awael."

### Message 4 :
"Génère le Chapitre 2 complet : Cadre théorique (innovation managériale, transformation digitale, Management 4.0)."

### Message 5 :
"Génère le Chapitre 3 complet : Diagnostic organisationnel."

### Message 6 :
"Génère le Chapitre 4 complet : Conception de la solution."

### Message 7 :
"Génère le Chapitre 5 complet : La plateforme ElAwael Digital Platform et ses modules."

### Message 8 :
"Génère le Chapitre 6 complet : Impact organisationnel et humain."

### Message 9 :
"Génère la Conclusion générale, la Bibliographie et les Annexes."

---

## Étape 4 — Assembler le rapport

Copie chaque partie générée dans un document Word ou Google Docs,
puis applique la mise en forme :
- Police Times New Roman 12pt
- Interligne 1,5
- Marges : haut/bas 2,5cm, gauche 3cm, droite 2,5cm
- Titres : 14pt Gras
- Pagination en bas de page

---

## CONSEIL IMPORTANT

Si Claude Chat perd le contexte entre les messages, commence par :
"Rappel : je rédige mon rapport PFE sur la transformation digitale 
à l'Association El Awael. Voici le contexte : [coller RESUME_PROJET.md]"
